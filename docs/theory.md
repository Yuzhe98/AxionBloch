# Theory

## Bloch equations

The nuclear spin magnetization {math}`\mathbf{M}` evolves according to

```{math}
\frac{d\mathbf{M}}{dt}
=
\gamma \mathbf{M} \times \mathbf{B}
- \frac{M_x \hat{x} + M_y \hat{y}}{T_2}
- \frac{(M_z - M_0)\hat{z}}{T_1}
```

## Axion-induced effective magnetic field

The axion field gradient produces an effective magnetic field

```{math}
\mathbf{B}_\mathrm{axion}
=
\frac{g_{aNN}}{\gamma} \nabla a(t)
```
